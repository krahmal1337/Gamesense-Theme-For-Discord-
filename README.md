# DiscordSense Theme

A revived and updated version of the old DiscordSense theme for Discord.

This project was originally based on an old and broken theme that no longer worked on modern Discord builds. I fixed compatibility issues, cleaned up the codebase, and restored most of the original appearance while keeping the overall aesthetic and feeling of the classic design.

## Features

* Restored compatibility with modern Discord versions
* Clean dark UI inspired by the original Gamesense style
* Updated and optimized CSS
* Improved readability and spacing
* Lightweight and simple setup
* Maintained original atmosphere and visuals as much as possible

## Current State

The theme is fully usable, but still work in progress.

Some visual details may slightly differ from the original version.
The font is also different for now, but it will be updated in future versions.

## Installation

1. Download the `.theme.css` file
2. Move it into your BetterDiscord themes folder
3. Enable the theme inside BetterDiscord settings
